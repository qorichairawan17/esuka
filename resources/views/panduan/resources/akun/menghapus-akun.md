#### Menghapus Akun

Berikut panduan menghapus akun Kamu pada aplikasi {{ config('app.name') }}.

---

1. **Buka halaman profil Kamu.** dengan cara klik foto profil sudut kanan atas.

    <img class="img-fluid mb-2" src="{{ asset('assets/images/panduan/akun/dashboard-profile.png') }}">

2. **Klik Tombol Hapus Akun pada profil Kamu**.

    <img class="img-fluid mb-2" src="{{ asset('assets/images/panduan/akun/delete-account.png') }}">

3. **Kemudian Kamu akan diminta konfirmasi persetujuan penghapus akun**, Harap berhati hati dalam penghapusan akun karena seluruh data kamu pada aplikasi akan dihapus secara permanen termasuk juga setiap riwayat pendaftaran surat kuasa Kamu. Jika kamu yakin isikan kalimat <b>hapus akun saya</b> untuk melanjutkan.
    
    <img class="img-fluid mb-2" src="{{ asset('assets/images/panduan/akun/confirm-delete-account.png') }}">

4. **Akun Kamu berhasil dihapus**. Seluruh informasi data kamu pada aplikasi dihapus secara permanen

    <img class="img-fluid mb-2" src="{{ asset('assets/images/panduan/akun/account-has-deleted.png') }}">
